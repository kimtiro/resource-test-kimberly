# Solstice Ranch Landing Page

Responsive landing page build for the Resource Test - Kimberly T. front-end development project.

The layout uses the exported Figma frame metadata (`1446 x 4261`) and embedded Figma package images as the primary visual reference.

## Tech Stack

- Vite
- Bootstrap 5
- Sass/SCSS
- Vercel serverless function for the contact form

## Local Setup

```bash
npm install
npm run dev
```

For deployed form validation, deploy to Vercel so `/api/contact` runs server-side.

## Build

```bash
npm run build
```
